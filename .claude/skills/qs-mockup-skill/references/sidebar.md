# 左侧伸缩栏规范

## 概述
左侧伸缩栏用于版本管理和页面切换，采用白色悬浮设计，不占用主内容区域宽度。

## 布局特性
- **定位**：`position: fixed` 悬浮在页面左侧
- **宽度**：展开 260px，收起 70px
- **高度**：100vh（全屏高度）
- **z-index**：100（确保在内容上方）
- **阴影**：`4px 0 16px rgba(0, 0, 0, 0.08)`（右侧边缘阴影）

## 动画效果
- **过渡时间**：0.35s
- **缓动函数**：`cubic-bezier(0.4, 0, 0.2, 1)`
- **应用对象**：宽度变化

## 组件结构

### 头部 (sidebar-header)
- **内容**：标题 + 切换按钮
- **标题文案**：英文版本管理 (Version Control)
- **字体**：600 weight, 13px, 颜色 #333
- **间距**：24px 上下，16px 左右
- **按钮样式**：
  - 背景：#f0f0f0
  - 悬停：#e8e8e8
  - 圆角：6px
  - 尺寸：16px 字体，6px 8px 内边距

### 内容区域 (sidebar-content)
- **滚动**：支持纵向滚动
- **滚动条**：自定义样式，宽度 6px，颜色 #ddd
- **内边距**：12px 上下，0 左右

### 导航项 (nav-item)
- **展开状态**：
  - 内边距：12px 16px
  - 字体：13px, 颜色 #666
  - 悬停背景：#f8f8f8
  - 悬停颜色：#333
  - 活跃背景：#f0f7ff
  - 活跃颜色：#1976d2
  - 活跃左边框：3px solid #1976d2

- **收起状态**：
  - 内边距：12px 8px
  - 文字隐藏：`font-size: 0`
  - 文本对齐：center
  - **不显示页面名称**

## 主内容区域适配
- **定位**：`position: fixed`
- **尺寸**：100% 宽度，100vh 高度
- **位置**：left: 0, top: 0
- **作用**：确保 mockup 宽度固定，不受伸缩栏影响

## 交互行为
1. 点击切换按钮：侧边栏在 260px 和 70px 之间切换
2. 点击导航项：
   - 移除其他项的 active 类
   - 添加当前项的 active 类
   - 加载对应的 HTML 文件到 iframe
3. 收起时：隐藏所有导航项的文字内容
